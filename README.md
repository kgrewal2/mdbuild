# mdbuild
Vim Plugin for live Markdown Editing (Preview in the browser)

### Requirements
- python
- [marked](https://github.com/markedjs/marked): markdown parser

# Usage
- Open your markdown file (xyz.md) in vim
- Type `:call MDbuild()`

It will generate a HTML file as xyz.html
